# 简介
此工程以后作为存储我开发的jqueryui 控件使用
## 工程使用说明
- 前提条件：安装node，全局安装npm（npm一般默认安装当安装node后），gulp，bower。
- 执行命令：
  
  ```bash
     $ bower install
     ```
     执行此命令后会根据bower.json 下载所需的第三方依赖（jquery,jquery-ui,angular,bootstrap)
  ```bash
     $ npm install
     ```
     执行此命令后下载npm，gulp所需插件（目前主要是下载gulp插件）
